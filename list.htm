Види сайтів:
  1. Статичні - це сайти, що не мають back-end, тобто не мають бази з даними. Вони створюється через:
      • HTML
      • CSS
      • інколи JS...
          1. Динамічні - це сайти, що мають back-end, тобто беруть дані з бази. Вони створюється через:
          • HTML
          • CSS
          • обов'язково JS
          • php чи ruby та інше
          • HTML 2
          • CSS 2
